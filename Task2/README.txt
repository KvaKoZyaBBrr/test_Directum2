Просмотр сотрудников производится в таблице слева сверху.
Количество столбцов не фиксировано. ФИО и дата рождения удалить нельзя. 
Добавить новые столбцы можно нажатием на столбец "..." (можно добавить кнопку "Редактировать столбцы"). Примерное редактирование представлено на рисунке слева снизу.
При выборе строки в таблице (лучше двойной клик) откроется форма "Сотрудник" с заполненными полями.
При нажатии на кнопку "+"/"Добавить сотрудника" откроется пустая форма "Сотрудник".
Форма "Сотрудник" (справа) исеет поля для ввода. Обязательные поля помечаются звездочкой и при сохранении незполненные обязательные поля должны подстветиться красным.
Чекбокс "Проживаю по месту регистрации" контролирует фрейм с данными по мету проживания. (логику лучше установить как "проживаю"=чек выбран=фрейм скрыт. "не проживаю"=чек снят=фрейм виден). Появление фрейма - разворачивание сверху вниз.
Для добавления места работы и члена семьи - нажимать на кнопку "+". При нажании добавляется новый фрейм (разворачивание сверху вниз). При нажатии кнопки "-" фрейм удаляется (сворачивание снизу вверх).
Поля на схеме лучше выровнять по ширине. Учесть разрешение экрана, размер формы.
Основной референс - формы 1с. Анимации и стили лучше использовать как в Bootstrap.
